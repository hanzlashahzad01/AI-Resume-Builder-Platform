# AI Resume Builder Platform (MERN)

Web SaaS to create professional resumes with AI-assisted content, modern templates, and PDF export.

- Frontend: React (Vite), Tailwind, React Router v6, React Hook Form + Zod, Zustand
- Backend: Node.js, Express.js, JWT auth, OpenAI API (pluggable mock)
- DB: MongoDB (Mongoose)
- Deployment: Frontend (Vercel), Backend (Render/Railway)

Scripts are defined in each app. See frontend and backend README sections for setup.
